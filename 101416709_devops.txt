The COMP 2156 - DevOps for System Administration educational program integrates development teams with operations teams to improve the software development lifecycle. The course introduces students to Jenkins alongside Docker and Git for process automation of testing alongside deployment and continuous integration. The automation tools provided by Jenkins enable straightforward execution of code-building and testing procedures with subsequent deployment tasks. Through the course students learn to put DevOps practices into action so they can speed up the delivery of reliable software systems through teamwork. The DevOps workflows receive version control and collaboration through Git functionality.
