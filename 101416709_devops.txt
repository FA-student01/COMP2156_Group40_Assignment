The COMP 2156 - DevOps for System Administration educational program integrates development teams with operations teams to improve the software development lifecycle. Students learn Jenkins as well as Docker and Git automation tools in this course because these tools enable automated testing along with deployment and continuous integration procedures. Through Jenkins automation students can simplify complex tasks that involve code-building and testing and deployment. DevOps practices formation becomes possible through students learning how to use teamwork methods for speeding up dependable software system deployment. The DevOps workflows benefit from Git because it offers version control together with collaborative features.
