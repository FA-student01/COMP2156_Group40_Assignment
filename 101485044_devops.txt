COMP3104- DevOps Course Overview
DevOps focuses on bridging the gap between development and operations through automation and collaboration.
The course introduces BASH scripting and CLI commands to help automate tasks and system administration.
Students learn about Version Control Systems (VCS) like Git and GitHub, which are essential for managing source code.
The curriculum covers monolithic vs. microservices architecture and the importance of scalability in applications.
DevOps roles, responsibilities, and tools such as Jenkins, Docker, and Kubernetes are explored.
The course emphasizes CI/CD pipelines, enabling continuous integration and deployment of software.
Topics include system administration skills that help transition into a DevOps career.
Build automation tools and web hosting concepts are introduced to streamline development workflows.
Students gain hands-on experience with Jenkins installation and infrastructure deployment.
By the end of the COMP3104 course, students will gain hands-on experience in DevOps tools, automation, and CI/CD implementation.
The DevOps course equips students with automation, CI/CD, and cloud computing skills to streamline software development and deployment.
